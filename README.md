# Dormly | Знайди свого іделаьного співжителя

<h2>Що таке Dormly</h2>

<p>Dormly — це вебплатформа, створена для студентів, які проживають у
гуртожитках або шукають житло поруч з університетом. Основна ідея проєкту
полягає в тому, щоб зробити процес пошуку сусідів та важливої побутової
інформації простим, зручним і доступним.</p>

<p>Dormly виступає своєрідним "цифровим помічником" студента-гуртожитчанина,
який дозволяє шукати потенційних сусідів, переглядаючи їх профілі та опис
вподобань;</p>


<h2>Суть Dormly</h2>

Суть Dormly полягає в тому, щоб:

<ul>
  <li> спростити життя студентам, які стикаються з хаосом і незручностями
під час заселення чи пошуку сусідів</li>
  <li>покращити комунікацію між студентами, створивши безпечний простір
із реальними профілями;</li>
  <li>оптимізувати процес заселення, дозволивши студентам робити
обдуманий вибір, а не випадковий.</li>
</ul>
